My Data Visualization Website

Data Collection

Over a five day period, I have collected data on my usage of Snapchat with the goal of defining my personal habits. My data collection spreadsheet does not simply track each individual time I open and use the app. Each instance of usage bears the following characteristics:

  • location
  • Activity
  • Intention
  • Format
  • Timespan
  • Tech Feature

These characteristics help to define trends in my usage that become apparent through my actual data chart which you will see in a moment. In simple terms, I want to know how certain situations effect how I use Snapchat, when I use it, and how often.

My Visualization

This chart allows me to identify trends between activities, locations, intentions, and technical features related to Snapchat usage. Color coated tallies indicate each of these four aspects over a five day period. By analyzing the tallies, I can see exactly when I use Snapchat, and in what specific circumstances that I use the application. From left to right I am able to discern patterns of color as well as the physical size of tally groupings to see how often Snapchat was used on a particular day, and the kind of aspects that follow its usage.
